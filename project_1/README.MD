Практика: 
* написать cli утилиту, которая принимает на вход директорию или файл в виде ./util ./path/to/dir или go run main.go ./path/to/dir
* сканирует переданную директорию или файл и считает общий вес содержимого
* после подсчета выводит в консоль общий вес данных в байтах
* добавить доп параметр, для отображения человекачитаемого размера в kb, mb, gb и т.п.

Для этого нужно будет сделать
* установить golang
* создать папку для проекта и выполнить в ней go mod init тут_имя_утилиты,
* создать main.go файл
* можно посмотреть (или спросить gpt) как обычно структурируют проекты в go
